# API 종류

## 레거시 API
* 예전부터 있던 봇 API에요.
* 자바스크립트, 커피스크립트, 비쥬얼 베이직으로 봇을 만들 때 지원하는 API 목록이 설명되어 있어요.

## API 2
* 봇 구동 앱 개발자들 사이에서 논의되던 표준안이에요.
* 자바스크립트로 봇을 만들 때 사용할 수 있어요.
