# 카카오링크
* 여기서는 `카카오링크 API`를 `비정상적인 방법으로 사용`하는 것을 다룰거에요.

## 정상적인 사용 방법
* 원래는 홈페이지나 애플리케이션에 있는 `카톡으로 공유하기` 버튼 같은거 누르면 카카오톡이 실행되고, 친구목록이나 채팅방 목록에서 공유할 곳을 누르면 카드 같은게 전송되는거에요.

## 비정상적인 사용방법
* 해당 API를 어케어케 잘 뚫어서 자동으로 보내게 만든 모듈을 사용하는거에요.
