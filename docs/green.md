# 그린스크립트

* `단순 자동응답` 기능을 기반으로 만들어진 유사 프로그래밍을 할 수 있는 스크립트 언어에요.
* 사용자의 숙련도 부족으로 인해 도배가 발생하기 아주 쉬운 기능으로, 그린스크립트를 사용하다가 생기는 모든 문제에 대한 책임은 사용자에게 있어요.
